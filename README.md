## Installation

1. First, install dependencies:

```bash
pip install numpy
pip install scikit-learn
pip install pywavelets
pip install seaborn


## virtual Environment [optional]

Windows:
    .venv\Scripts\activate
    .venv\Scripts\python

Unix/macOS:
    source .venv/bin/activate
    .venv/bin/python
```
